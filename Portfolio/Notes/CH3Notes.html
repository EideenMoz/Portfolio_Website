The CSS Box Model - The content box contains the content added using the HTML tags - The padding box surrounds the content and adds space inside of the element - The border box surrounds the padding area - The margin box surrounds the entire element https://dev.to/alvaromontoro/the-css-box-model-built-only-with-css-36m6 Width and Height - Set the width and height of the content box - Used with keywords, and values - Minimum/maximum values can be set with min-width, max-width, min-height, and max-height Padding - Defines the space inside an element and around the content box - Used with or values - Set padding to a 0 value to remove default styles - Can be defined using one to four values - Shorthand i.e. /* Same on all sides */ padding: 20px;
